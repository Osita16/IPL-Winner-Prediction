## 🧠 How the Model Thinks

The prediction is based on match dynamics like:

* 🏏 Batting Team
* 🎯 Bowling Team
* 📍 City
* 🎯 Target
* 📊 Current Score
* ⏱️ Overs Completed
* ❌ Wickets Lost

It processes these inputs through a trained ML pipeline and outputs the **winning probability**.
